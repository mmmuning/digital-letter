# 💌 Digital Letter Project

A personalized, interactive web application featuring a vintage-inspired digital letter, animations, and a special reveal.

## ✨ Features

- **Interactive Envelope:** A CSS-animated envelope that opens upon clicking the wax seal.
- **Integrated Media:** Custom audio player styled as a vintage cassette tape and embedded Spotify playlists.
- **Dynamic Animations:** Floating sparkle effects, handwritten signature styles, and smooth transitions.
- **Interactive Proposal:** A "Yes/No" interaction that leads to a final calendar reveal or a graceful exit.

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for the letter and form.
- **CSS3:** Custom animations, Flexbox/Grid layouts, and responsive design.
- **JavaScript (Vanilla):** Event listeners, DOM manipulation, and audio control logic.

## 📂 Project Structure

- `index.html` - The main structure of the letter.
- `style.css` - All custom styling and animations.
- `script.js` - Interaction logic for the buttons and audio.
- `/assets` - Contains images (`tape.jpg`, `sparkles.gif`) and audio (`recording.mp3`).
